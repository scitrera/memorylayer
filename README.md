# memorylayer.ai Open Source Repository

This is a monorepo layout for the open source components of memorylayer.ai.

The following packages are part of the open source repository:

- [memorylayer-core-python](./memorylayer-core-python): Python FastAPI Server for MemoryLayer.ai
- [memorylayer-sdk-python](./memorylayer-sdk-python): Python Client SDK for MemoryLayer.ai
- [memorylayer-sdk-typescript](./memorylayer-sdk-typescript): TypeScript Client SDK for MemoryLayer.ai
- [memorylayer-mcp-typescript](./memorylayer-mcp-typescript): TypeScript MCP Server for MemoryLayer.ai
- [memorylayer-sdk-langchain-python](./memorylayer-sdk-langchain-python): LangChain Python SDK for MemoryLayer.ai
- [memorylayer-sdk-llamaindex-python](./memorylayer-sdk-llamaindex-python): LlamaIndex Python SDK for MemoryLayer.ai
- [memorylayer-cc-plugin](./memorylayer-cc-plugin): MemoryLayer.ai Claude Code Plugin
