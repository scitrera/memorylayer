# MemoryLayer Explorer

Web UI for exploring MemoryLayer memories and graph relationships.

## Features

- Browse and search memories with filtering by type, subtype, tags, and importance
- Visualize memory graphs and associations using interactive node-edge diagrams
- Walk relationship paths between memories
- Inspect sessions and working memory
- Analytics dashboard with memory distribution and activity charts

## Getting Started

1. Copy `.env.example` to `.env.local` and configure your MemoryLayer server URL
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Open [http://localhost:3100](http://localhost:3100)

## Requirements

- Node.js >= 18
- A running MemoryLayer server (default: http://localhost:61001)
