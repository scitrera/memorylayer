## Quick Install

```bash
# Server
pip install memorylayer-server

# Python SDK
pip install memorylayer-client

# TypeScript SDK
npm install @scitrera/memorylayer-sdk

# MCP Server (for Claude, etc.)
npm install @scitrera/memorylayer-mcp-server
```
